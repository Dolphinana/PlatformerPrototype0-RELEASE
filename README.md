# CONTROLS:

Move: wasd or arrow keys

Jump: Z, K or SPACE

Shoot: X or J

Reset: R

Only reset back to spawnpoint: T
  
Change levels: numbers

Edit levels: Mouse

Exit game: ESCAPE 2x
